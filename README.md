# 🚀 ChamaAí

ChamaAí é uma plataforma que conecta clientes a profissionais confiáveis de forma rápida e simples.

A proposta é facilitar a busca por serviços locais como:

- Eletricista
- Encanador
- Pedreiro
- Diarista
- Técnico de informática
- Pintor
- Montador de móveis

## 🎯 Objetivo do Projeto

Criar um sistema moderno, intuitivo e escalável, onde:

### Cliente poderá:
- Criar conta
- Buscar profissionais
- Filtrar por categoria
- Solicitar serviços
- Avaliar atendimento

### Profissional poderá:
- Criar perfil
- Informar serviços prestados
- Receber solicitações
- Gerenciar pedidos
- Construir reputação
